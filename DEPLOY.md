# Sistema de Agenda para Salão de Beleza

## Deploy Online

Para compartilhar o sistema online, siga estes passos:

### Opção 1: Vercel (Recomendado)
1. Acesse [vercel.com](https://vercel.com)
2. Crie uma conta gratuita
3. Conecte com GitHub
4. Faça upload do projeto
5. Deploy automático

### Opção 2: Netlify
1. Acesse [netlify.com](https://netlify.com)
2. Crie uma conta gratuita
3. Arraste a pasta do projeto
4. URL instantânea

### Opção 3: Ngrok (Temporário)
1. Registre-se em [ngrok.com](https://ngrok.com)
2. Instale: `npm install -g ngrok`
3. Configure: `ngrok config add-authtoken SEU_TOKEN`
4. Execute: `ngrok http 3001`
5. Compartilhe a URL pública gerada
